NOTES
-----

1. WDF Primitive Families (Diode, BJT, etc.) are implemented as separate
   header files with constexpr preset structs, not merged into the main
   nonlinear headers. This allows circuits to choose specific transistor
   models at compile time without code duplication.

2. The three completed effects (Big Muff, Soft Focus, and their future chain
   "Shoegaze") represent substantial WDF + DSP integration milestones.
   Combined, they demonstrate:
     - Multi-stage transistor amplifier synthesis
     - Nonlinear harmonic complexity
     - Granular+reverb processing (exotic effect chains)
     - Parameter-driven circuit variants

3. Test audit (2026-03-25) identified two regressions in WDF solvers.
   Both are within scope of the WDF audit/repair prompt (2026-03-25).
   Estimated fix complexity: Jacobian matrix correction (PnP BJT) and
   adaptor tree scattering order (Wah). No architectural changes needed.

4. FDN reverb primitives are already partially in place (CombFilter,
   Hadamard4). The 2026-03-18 prompt added the extended Hadamard8,
   ModulatedAllpass, BitCrusher, and full FdnReverb<N> template.
   External RAM buffer management is critical (allocate at init time,
   not during audio callback).

5. All new implementations must respect hard constraints:
     - No heap allocation (no new, malloc, std::vector)
     - No exceptions
     - No RTTI
     - Single-precision float only (1.0f, not 1.0)
     - C++20 standard
     - noexcept on all audio-path methods
     - Cycle budget: ~15,000 cycles/sample total
